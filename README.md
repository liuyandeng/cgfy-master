### 2.[cgfy-lock](https://blog.csdn.net/qq877507054/article/details/107610780)

### 3.[cgfy-rabbitmq](https://blog.csdn.net/qq877507054/article/details/102967445)

### 4.cgfy-transaction


事务的相关概念,DB+MQ

### 5.cgfy-libreoffice
