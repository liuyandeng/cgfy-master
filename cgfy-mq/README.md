# rabbitmq

#### 介绍
消息中间件rabbitmq测试


#### 安装

1.安装Erlang 

2.安装RabbitMQ

3.在rabbitmq的安装目录下找到sbin目录，进入并在此目录打开cmd窗口(进入sbin的cmd窗口),
输入rabbitmq-plugins enable rabbitmq_management (需要重启服务)


4.输入:http://localhost:15672, 浏览器访问,  用户名/密码 guest/guest
